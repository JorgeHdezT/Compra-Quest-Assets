# Tienda Assets para Compra Quest
Este repositorio contiene todos los **assets de la tienda** de Compra Quest, incluyendo **skins**, **mascotas** y **canciones**, junto con los archivos JSON que describen cada categoría.  
Se utiliza con **CDN jsDelivr** para cargar los datos y recursos dinámicamente sin necesidad de hacer build cada vez que se agregan nuevos elementos.
